
# Shobdo

Modernized migration of the original Shobdo project.

## What was preserved
- Original visual direction
- Original styling structure
- Original asset structure
- Music archive/editorial layout style

## What was upgraded
- Next.js 14 App Router
- Cleaner component structure
- Easier scalability
- Better responsive behavior
- Reusable sections/components
- Cleaner naming conventions

## Run

npm install
npm run dev
